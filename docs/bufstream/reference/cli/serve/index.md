# bufstream serve

Serve a Bufstream cluster

### Usage

```console
$ bufstream serve [flags]
```

### Description

A process that serves a Bufstream cluster.

### Flags

#### \-c, --config _string_

Path to the config file.

#### \-h, --help

help for serve

#### \--inmemory

Use in-memory storage (for development only).
